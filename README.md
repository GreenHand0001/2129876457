# 项目简介

## 核心功能
- 提供图形化界面，实现点击即训练的零门槛YOLO训练体验
- 内置数据集向导，帮助用户快速完成数据集配置
- 智能参数优化功能，避免新手常见训练陷阱
- 支持检测/分割/分类/关键点四大任务类型
- 实时监控训练进度和效果曲线
- 智能内存管理，确保训练过程稳定
- 完整日志记录，便于问题排查
- 一键停止恢复功能，灵活控制训练过程

## 适用场景
- 学习研究：适用于毕业设计、课程作业、算法实验等场景
- 商业应用：可用于缺陷检测、安防识别、医疗诊断等专业领域
- 快速验证：支持原型开发、数据集测试、模型对比等需求

## 技术特点
- 基于YOLOv8官方引擎，确保训练效果的专业性
- 简化传统命令行操作，降低使用门槛
- 优化内存管理机制，提高训练稳定性
